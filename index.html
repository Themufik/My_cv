Quark CRM Service
Разработка REST API микросервиса на FastAPI для учета клиентов с интегрированной системой безопасности. Реализована JWT-авторизация пользователей (роутер auth.py) для защиты эндпоинтов. Настроена модульная структура приложения с разделением логики по APIRouter, строгая валидация типов через Pydantic-схемы и автоматическое управление сессиями базы данных SQLite с помощью SQLAlchemy ORM.

// backend · web application
Areon Platform
Веб-платформа на Django для публикации и обмена образовательными курсами. Спроектирована реляционная структура БД (Django ORM), связывающая профили пользователей, категории, курсы и текстовый контент лекций. Логика приложения реализована через классические Server-Side Views, а пользовательский интерфейс — на базе динамических шаблонов Django Templates. Для администрирования, модерации контента и управления правами доступа настроена стандартная административная панель.

// async core · telegram bot
Nova: Calorie Tracker
Асинхронный Telegram-бот на aiogram 3.x для быстрого логирования съеденных калорий. Реализован парсинг текстовых сообщений (название и вес через пробел), калькуляция калорийности порций на основе базы данных продуктов и ежедневный учет через SQL-агрегатор SUM(). Данные сессий и логов сохраняются в локальную БД через aiosqlite. Также добавлена команда расчета ИМТ и вывода базовых рекомендаций по калоражу. Конфигурация проекта скрыта в .env.
